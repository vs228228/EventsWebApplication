Доброго времени суток. Инструкция по запуску проекта + аккаунты для теста
Для запуска проекта на компьютере должны быть: .net8, asp.net, angular, node.js
Скачиваем себе проект и открывает решение. Ждём, пока все пакеты npm подкачаются.
Когда установка завершилась, открываем в терминале папку EventsWebApplication.client и в ней прописываем:
npm install
npm init -y
npm run build
Затем запускаем решение(сервер)
Снова в папке client прописываем ng serve
Теперь у вас открыт swagger, где можно посмотреть все http методы. Для запуска визуала необходимо зайти в http://localhost:4200/ (сервер на 7059 порте стоит вдруг что)
